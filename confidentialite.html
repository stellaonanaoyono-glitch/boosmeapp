2. Données collectées

2.1 Données d'inscription

Prénom, nom (optionnel), adresse email, pays de résidence, langue préférée, mot de passe (hashé, non accessible).

2.2 Données de profil IA

Tranche d'âge, situation professionnelle, disponibilité quotidienne, objectif principal. Ces données sont utilisées uniquement pour personnaliser le contenu des challenges.

2.3 Données de progression

Challenges commencés et terminés, jours validés, scores, streak, certificats obtenus, réflexions personnelles (optionnel).

2.4 Données de paiement

Les données de paiement sont traitées directement par nos prestataires (NotchPay, Stripe). BOOST.ME ne stocke jamais les numéros de carte ou les informations Mobile Money complètes — seulement la référence de transaction et le statut.

2.5 Données techniques

Adresse IP, type de navigateur, système d'exploitation, pages visitées, durée de session. Ces données sont collectées automatiquement à des fins de sécurité et d'amélioration du service.

3. Finalités du traitement

  • Gestion du compte : création, authentification, sécurité
  • Fourniture du service : accès aux challenges, personnalisation IA, suivi de progression
  • Paiements : traitement des transactions, facturation, remboursements
  • Communications : emails de confirmation, rappels quotidiens, notifications de la communauté (selon préférences)
  • Amélioration du service : analyses statistiques agrégées et anonymisées
  • Obligations légales : conformité fiscale et comptable

4. Base légale des traitements

  • Exécution du contrat (inscription, accès aux challenges, paiements)
  • Consentement (communications marketing, cookies non essentiels)
  • Intérêt légitime (sécurité, prévention de la fraude, amélioration du service)
  • Obligation légale (conservation des factures)

5. Durée de conservation

  • Données de compte : durée de l'abonnement + 3 ans après clôture
  • Données de progression : durée de l'abonnement, supprimées à la clôture sauf demande contraire
  • Données de paiement : 10 ans (obligation comptable)
  • Données techniques : 13 mois maximum

6. Partage des données

BOOST.ME ne vend jamais vos données personnelles. Elles peuvent être partagées uniquement avec :

  • Supabase Inc. — hébergement base de données (serveurs UE)
  • Netlify Inc. — hébergement plateforme
  • Brevo (ex-Sendinblue) — envoi d'emails transactionnels
  • NotchPay / Stripe — traitement des paiements
  • Groq Inc. — personnalisation IA (données de profil anonymisées)

Tous ces prestataires sont soumis à des obligations contractuelles strictes de confidentialité.

7. Vos droits

Conformément à la réglementation applicable, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : demander la suppression de votre compte et données
  • Droit à la portabilité : récupérer vos données dans un format structuré
  • Droit d'opposition : vous opposer à certains traitements
  • Droit de retrait du consentement : à tout moment pour les traitements basés sur le consentement

Pour exercer ces droits : boostmeapp@gmail.com. Nous répondons dans un délai de 30 jours.

8. Sécurité des données

Nous mettons en œuvre des mesures techniques et organisationnelles pour protéger vos données :

  • Chiffrement HTTPS/TLS sur toutes les communications
  • Mots de passe hashés (bcrypt) — jamais stockés en clair
  • Authentification sécurisée via Supabase Auth
  • Clés API jamais exposées côté client
  • Row Level Security (RLS) sur la base de données
  • Accès aux données restreint au personnel autorisé

9. Intelligence artificielle et données personnelles

Les données de votre profil (tranche d'âge, objectif, disponibilité) sont transmises à l'API Groq pour générer des adaptations personnalisées. Ces transmissions sont anonymisées — aucun identifiant personnel (email, nom) n'est envoyé à l'IA. Groq s'engage contractuellement à ne pas utiliser ces données pour entraîner ses modèles.

10. Contact et réclamations

Pour toute question relative à cette politique : boostmeapp@gmail.com

Si vous estimez que vos droits ne sont pas respectés, vous pouvez saisir l'autorité de protection des données compétente dans votre pays de résidence.
